# Typescript Library

This is a typescript library built with `ts-lib-starter`. [Make yours now]('https://github.com/youssef-ahamid/ts-lib-starter/')

## Installation

```bash
npm i <package> #npm
yarn add <package> #yarn
pnpm add <package> #pnpm
```

## Usage

```typescript
import package from '<package>';
package.doSomething();
```
